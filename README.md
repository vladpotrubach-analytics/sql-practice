## SQL Practice Repository

This repository contains **learning and practice SQL tasks**
focused on data and product analytics.

### Topics covered:
- Basic queries and aggregations
- JOINs
- Subqueries and CTEs
- Window functions
- Simple business analytics cases

### Purpose:
- Practice SQL syntax
- Learn how SQL answers business questions
- Prepare for junior data / product analyst interviews

The solutions are based on learning materials, practice tasks,
and guided examples.
